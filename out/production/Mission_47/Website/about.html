About
Intended for audiences 13 years of age and older, Mission 47 is an interactive text-based adventure game, in which you must journey to save their crew from execution by a nefarious Tilani warlord.
Developer
I am an aspiring software engineer attending Georgia Gwinnett College. I developed this Java program in Software Development I at Georgia Gwinnett College
Installation
No installation required. Users can play the game using a Java IDE or their operating system's built-in terminal, such as Windows PowerShell for Windows OS users or Terminal for macOS users. The user can read the user manual through a text editor, such as Notepad or Notepad++.
